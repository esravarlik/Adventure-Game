# Places:
- Safe House
 - Feature: Regenerating Health
- Cave
 - Monster : Zombie (1-3)
 - Attribute: War + Loot
 - Item: Food (Food)
- Forest
 - Monster: Vampire (1-3 Pieces)
 - Attribute: War + Loot 
 - Item: Wood (Firewood)
- River
 - Monster : Bear (1-3 Pieces)
 - Attribute: War + Loot
 - Item: Water (Water)
- ToolStore
 - Feature: Buying Supporting Items
 - Weapon: Pistol, Sword, Rifle
 - Armor: Light, Medium, Heavy
